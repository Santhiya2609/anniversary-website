## 📞 Support

### If Something Doesn't Work:

1. **Check Console:** Press F12 → Console → Look for errors
2. **Check Paths:** Verify image/video/audio file paths
3. **Check Syntax:** Ensure quotes/brackets/commas are correct
4. **Read Docs:** TEMPLATES.md has working examples
5. **Test Locally:** Use Python's http.server, not just open
6. **Clear Cache:** Ctrl+Shift+Delete in browser

### Common Issues & Fixes:
- **Videos won't play** → Use MP4 format
- **Music won't play** → Check audio folder paths
- **Images won't show** → Verify image paths and folders
- **Dark mode broken** → Clear localStorage
- **Quiz won't work** → Check JavaScript syntax
